## ✨ 주요 기능 (Key Features)

* **🔄 구간 반복 (AB Loop):** 원하는 유튜브 링크와 시작초, 종료초를 입력하면 해당 구간만 무한 반복 재생됩니다.
* **🩳 쇼츠(Shorts) & 라이브 지원:** 일반 영상은 물론, 최신 유튜브 쇼츠 링크와 라이브 스트리밍 주소까지 완벽하게 인식합니다. [✨ New]
* **📝 실시간 메모장:** 영상 오른쪽에 나만의 노트를 작성할 수 있으며, 입력 후 다른 곳을 클릭하면 자동으로 저장됩니다.
* **👯 구간 복제 기능:** 마음에 드는 영상과 메모 내용을 버튼 하나로 통째로 복제하여 원본 바로 아래 칸에 생성할 수 있습니다. [✨ New]
* **🛠️ 실시간 시간 수정:** 구간을 다시 설정하고 싶을 때, 삭제할 필요 없이 '시간 수정' 버튼으로 즉시 조절 가능합니다.
* **🌙 다크모드 지원:** 밤이나 어두운 환경에서도 눈이 편안하게 공부할 수 있도록 원클릭 다크모드를 지원합니다.
* **☁️ Firebase 동기화:** 데이터베이스 연동으로 새로고침을 하거나 다른 기기에서 접속해도 내가 저장한 리스트가 그대로 유지됩니다.
* **🛡️ 1MB 데이터 안전 장치:** 프론트엔드단에서 데이터 용량을 사전에 체크하여 브라우저나 DB가 뻗는 현상을 안전하게 방지합니다. [✨ New]
* **🎨 반응형 UI & 경기천년체:** 모바일 뷰포트 최적화로 스마트폰에서도 완벽하게 작동하며, 가독성 높은 웹 폰트를 적용했습니다. [✨ Updated]

## 📂 프로젝트 구조 (Project Structure)

코드가 길어짐에 따라 유지보수가 쉽도록 **관심사 분리(Separation of Concerns)**를 적용해 파일을 구조화했습니다.

```text
my-ytube-app/
├── 📄 index.html  # 모바일 뷰포트 및 웹사이트의 뼈대를 담당하는 HTML
├── 🎨 style.css   # 다크모드, 웹 폰트(경기천년체) 및 반응형 레이아웃을 담당하는 CSS
└── ⚡ script.js   # 유튜브 API, 정규식 파싱, 복제 로직 및 Firebase 동기화를 담당하는 JS
